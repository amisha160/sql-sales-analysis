SQL Sales Analysis Project

Overview
This project demonstrates SQL-based data analysis using MySQL. It includes database creation, data insertion, and analytical queries to derive business insights.

Features
- Sales aggregation
- Profit analysis
- Customer analysis
- Loss identification
- Category-wise insights

Insights
- Technology category generated high profit
- Some transactions resulted in losses
- Top customers contribute significantly to revenue

Tools Used
- MySQL
- SQL (Joins, Aggregation, Group By)

Skills Demonstrated
- Data querying
- Business analysis
- SQL optimization basics
